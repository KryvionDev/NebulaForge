# NebulaForge ⚙️

Uma mini engine 3D feita para rodar direto no navegador.  
Foco: simplicidade, performance e integração com MongoDB para perfis e projetos.

## Recursos
- Editor web leve (HTML + JS)
- Renderização WebGL2
- Física simples (posição, velocidade)
- Sistema de contas com JWT e MongoDB
- Exportação básica de cenas

## Rodando
```bash
npm install
npm run server  # Inicia API
npm run dev     # Inicia frontend
